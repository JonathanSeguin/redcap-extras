Welcome to REDCap Extras

This is the public website for the REDCap Extras GitHub repository. The content on this site is intended for REDCap administrators and server administrators interested in adding features to REDCap via Plugins, Hooks, or API code.

Access

While this website is publically available, the actual repository is currently private. In order to gain access to the repo you must complete an application request and be added to the group. The decision to make the initial repository private was to ensure that un-vetted and un-tested code was not deployed to any REDCap servers. We hope that by working together, as first as a private group, we will eventually be able to release code which can be made publically available.

What's New

6/19/2014
I'm pushing out an initial release of the imagemap hook (compatible with version 5.10 and up). This allows you to insert an imagemap question into a REDCap survey. The imagemap selections can be saved as either a text question (comma-separated values) or as a checkbox question. For additional details, please see the imagemap hook.
